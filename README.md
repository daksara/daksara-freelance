# 🎬 Subtitle Maker — Panduan AI Workflow

Panduan lengkap cara kerja sebagai **Subtitle Maker freelance** menggunakan AI (Turboscribe & Transkriptor) dari HP. Cocok untuk pemula yang mau mulai kerja remote bergaji dollar.

## 🚀 Cara Pakai

1. Download/clone repo ini
2. Buka file `index.html` di browser HP atau laptop
3. Pilih platform (Turboscribe / Transkriptor)
4. Ikuti langkah-langkah dari awal sampai selesai
5. Klik tombol untuk langsung ke website yang dibutuhkan

## 📋 Isi Panduan

- ✅ Alur kerja lengkap dari audio → file .srt siap kirim
- ✅ Step-by-step Turboscribe (gratis 3 file/hari)
- ✅ Step-by-step Transkriptor (support YouTube URL)
- ✅ Tabel perbandingan kedua platform
- ✅ Tips pro untuk hasil lebih cepat & akurat
- ✅ Tombol langsung ke semua tools yang dibutuhkan
