🌀 Turboscribe.ai

// Gratis 3 file/hari · Akurasi tinggi · Paling mudah untuk pemula

Buka Turboscribe
01

Buka & Daftar Akun Gratis

Pergi ke turboscribe.ai → klik "Get Started Free" → daftar pakai email atau Google. Tidak perlu kartu kredit.

GRATIS
02

Upload File Audio / Video

Klik "Transcribe" → drag & drop atau pilih file dari HP. Format yang didukung: MP3, MP4, WAV, M4A, WebM. Maks 30 menit per file (gratis).

TIPS: File bersih = hasil lebih akurat
↗ Upload Sekarang
03

Pilih Bahasa & Mode

Pilih bahasa sumber video (English, Indonesian, dll). Mode "Whale" = paling akurat tapi lebih lama. Mode "Dolphin" = lebih cepat untuk draft awal.

TIPS: Pilih bahasa manual, jangan "Auto Detect" untuk akurasi maksimal
04

Tunggu Proses Transkripsi

Video 10 menit → sekitar 1–2 menit proses. Video 30 menit → sekitar 3–5 menit. Kamu bisa tinggalkan halaman, nanti balik lagi.

05

Review & Edit Hasil

Hasil muncul di editor Turboscribe. Klik teks untuk edit langsung. Perhatikan: nama orang, istilah teknis, dan kata-kata yang ambigu sering salah → koreksi manual.

CEK: Nama brand, istilah khusus, singkatan
06

Export sebagai File .SRT

Klik tombol "Export" → pilih format "SRT" (untuk subtitle) atau "TXT" (untuk teks biasa). File langsung terdownload ke HP kamu.

FORMAT .SRT = SIAP KIRIM KE KLIEN
07

Cek Timing di Subtitle Edit Online

Buka Subtitle Edit Online → paste isi .srt kamu → play video sambil cek apakah subtitle sync. Sesuaikan timing kalau ada yang terlambat atau terlalu cepat.

↗ Buka Editor
08

Kirim ke Klien ✅

Upload file .srt ke Google Drive / WeTransfer / langsung di platform (Fiverr, Upwork, Rev). Tandai pekerjaan selesai dan minta review bintang 5!

💰 DONE — TUNGGU PEMBAYARAN

// Tips Pro Turboscribe

  • Pecah video panjang jadi potongan 15 menit supaya tidak habis kuota gratis harian
  • Gunakan fitur "Speaker Detection" kalau video punya lebih dari 1 pembicara
  • Upgrade ke $10/bulan untuk unlimited file — worth it kalau sudah dapat klien rutin
  • Hasil Turboscribe bisa langsung dipaste ke Google Docs untuk format alternatif
  • Simpan semua hasil di folder Google Drive biar rapi dan mudah dicari klien lagi

🎙️ Transkriptor.com

// Trial gratis · Support 100+ bahasa · Export langsung .SRT/.VTT

Buka Transkriptor
01

Daftar & Login

Buka transkriptor.com → klik "Start for Free" → daftar pakai Google atau email. Kamu dapat trial gratis untuk beberapa file pertama.

TRIAL GRATIS TERSEDIA
↗ Daftar Gratis
02

Masuk ke Dashboard

Setelah login, kamu akan masuk ke dashboard utama. Di sini kamu bisa lihat semua file yang sudah ditranskripsi dan memulai transkripsi baru.

03

Upload File atau Paste Link YouTube

Klik "+ New" atau "Upload" → pilih file dari HP kamu. Atau langsung paste link YouTube / Google Drive. Format yang didukung: MP3, MP4, WAV, M4A, dan banyak lagi.

TIPS: Link YouTube lebih cepat daripada upload manual
↗ Upload File
04

Pilih Bahasa Sumber

Transkriptor support 100+ bahasa. Pilih bahasa yang digunakan dalam video. Kalau video campur dua bahasa, pilih bahasa yang dominan.

PENTING: Jangan pakai Auto-Detect untuk video Indonesia
05

Mulai Transkripsi & Tunggu

Klik "Transcribe" → Transkriptor akan memproses file kamu. Waktu proses tergantung panjang video. Kamu akan dapat notifikasi email saat selesai.

06

Edit di Built-in Editor

Transkriptor punya editor bawaan yang bagus. Kamu bisa klik setiap kata untuk koreksi, putar audio langsung di editor, dan lihat timestamp per kata untuk cek timing.

FITUR: Klik kata = langsung loncat ke bagian audio itu
07

Export Format Subtitle

Klik tombol "Export" → pilih format: SRT (paling umum), VTT (untuk web), atau TXT. File langsung download ke HP. SRT adalah format yang paling dibutuhkan klien.

FORMAT: .SRT untuk YouTube, Vimeo, Upwork · .VTT untuk web
08

Cek Timing Final

Buka Subtitle Edit Online → import file .srt → play video sambil cek subtitle. Pastikan tidak ada yang terlambat muncul atau terlalu cepat hilang.

↗ Cek Timing
09

Kirim ke Klien & Tagih Bayaran ✅

Kirim file .srt via platform kerja (Fiverr/Upwork/Rev) atau langsung ke klien. Tandai order selesai dan minta review. Selamat, kamu baru dapat dollar pertama!

💰 PENGHASILAN PERTAMA MENANTI

// Tips Pro Transkriptor

  • Fitur "YouTube Link" sangat berguna — tidak perlu download video dulu, langsung paste URL
  • Transkriptor bisa detect speaker otomatis — bagus untuk podcast atau interview
  • Gunakan fitur search & replace untuk ganti nama yang salah di seluruh dokumen sekaligus
  • Simpan template setting bahasa favorit supaya tidak perlu set ulang tiap kali
  • Coba integrasi dengan Google Drive untuk simpan hasil otomatis ke cloud
